## What We Do

SIINC provides automated, independent version backups for Common Data Environments (CDEs) used in the construction industry. We protect critical project data with enterprise-grade backup and recovery capabilities.

### Primary Integrations
- Autodesk Construction Cloud (ACC)
- BIM 360
- Autodesk Build
- Other Common Data Environments

### Key Features
1. **Scheduled Backups**: Continuous incremental protection with customizable schedules
2. **Granular Restore**: Recover by file, folder, or entire project instantly
3. **Compliance Reports**: ISO/cyber-insurance aligned documentation with immutable audit logs
4. **BYO Storage**: Azure, AWS, S3-compatible, or on-premises storage options
5. **Native Integration**: Direct CDE integration with OAuth - no agents required
6. **Full Metadata**: Preserve all versions, permissions, and audit trails
